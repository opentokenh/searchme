# Weekly Review Template

## 文件路径
`research/weekly-reviews/YYYY-WXX.md`

## 完整模板

```markdown
# YYYY-WXX 周回顾

## 📊 本周完成

### 论文阅读
- [论文1标题] — 1-2句核心贡献
- [论文2标题] — 1-2句核心贡献

### 实验进展
- [实验名称]：结果如何？

### 想法与发现
- 有没有新的insights或连接点？

---

## 💡 本周最大收获
（1句话，不要超过20字）

---

## 😤 卡点
- 具体卡在哪里？
- 需要什么帮助？

---

## 🎯 下周最重要的一件事
（只选1件）

---

## 📅 下周计划
- [ ] 
- [ ] 
```

## 快速自检清单

- [ ] 对照 `research/interests.md` 检查方向优先级
- [ ] 检查 `research/connections.md` 有无更新
- [ ] 检查 `research/ideas.md` 有无新想法
- [ ] 是否有 cron 任务产出需要回顾？
