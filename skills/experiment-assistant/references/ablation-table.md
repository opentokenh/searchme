# Ablation Study Table

每次 ablation study 使用以下格式：

```
| 组件/参数 | 去掉? | 结果变化 | 结论 |
|-----------|------|----------|------|
| 基准 | — | 基线 | 对比基准 |
| +组件A | 否 | ↑X% | 有效 |
| +组件B | 否 | ↓Y% | 负向效果 |
| -组件A | 是 | ↓Z% | 关键组件 |
```

## 典型失败模式

| 模式 | 描述 | 检查方法 |
|------|------|----------|
| 数据泄漏 | train/test混淆 | 检查split |
| 随机种子 | 结果不可复现 | 多次run取平均 |
| 超参过拟合 | 调参在test上 | 用validation set |
| 度量错误 | metric计算错误 | 独立验证代码 |
| 偷看结论 | 多次实验挑好的 | 预定分析计划 |
