# Connection Brief Template

每次 Innovation Hunter 产出时，使用以下格式：

```
== 连接确认 ==
[1-2句话描述]

== 为什么有意思 ==
[这个方向值得探索的原因，
 不是已经在做的事，
 而是我们独特能做的事]

== 最大风险 ==
[最大的失败可能性]

== 需要的条件 ==
[如果要尝试，需要什么？
 数据？算力？合作？]

== 相关论文 ==
[如果有，给出 2-3 篇]
[如果没有，写"需要进一步调研"]
```

## 示例

```
== 连接确认 ==
用多模态Agent作为遥感图像的主动理解中枢，
配合北斗定位信号做实时的视觉导航增强。

== 为什么有意思 ==
传统遥感图像解读是被动的"图→描述"，
但结合北斗的精确定位和时间戳，
可以做到"定位→感知→推理"的闭环。
这是卫星视觉和导航领域都没有在做的事。

== 最大风险 ==
多模态模型在卫星图像上的泛化能力不足，
训练数据难以获取。

== 需要的条件 ==
- 高分辨率遥感图像数据集（Sentinel-2 / 高分系列）
- 北斗定位轨迹数据
- 微调算力

== 相关论文 ==
- "RSVLM: Remote Sensing Vision Language Model" (arXiv:XXXX)
- "GNSS-Visual Inertial Odometry" (ICRA 2024)
- 需要进一步调研
```
