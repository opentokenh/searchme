## 高级搜索技巧

- `after:2025-01-01` — 限发布时间
- `cat:cs.CV` — 限cs.CV分类
- `all:exact phrase` — 精确匹配词组
- `OR` — 逻辑或

## 快速找最新热文

```
site:arxiv.org/abs 2026    # 今年
site:arxiv.org/new/        # 今日新提交
```
